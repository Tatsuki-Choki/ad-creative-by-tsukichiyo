## ゴールデンルール

### ルール1: 再生成より編集

モデルは反復的な改善に優れている。完全に新しい画像を要求するのではなく、既存の出力に対して具体的な修正を指示する。

```
✕ 「夕日の写真を作って」（ゼロから再生成）
○ 「ライティングを夕日に変えて、テキストをネオンブルーにして」（編集）
```

### ルール2: 自然な文章で指示

キーワードの羅列ではなく、人間のクリエイティブディレクターに説明するように完全な文章で書く。

```
✕ 「犬, 公園, 4k, リアル」
○ 「公園の芝生の上でフリスビーをキャッチしようとジャンプしているゴールデンレトリバー。
    午後の柔らかい日差しが犬の毛並みを金色に照らしている。」
```

### ルール3: 具体的に描写

汎用的な表現を避け、精密なディテールを含める。質感の指定も効果的。

```
✕ 「女性のポートレート」
○ 「ヴィンテージのシャネル風スーツを着た、品のある年配の女性。
    マットな質感の肌、シルバーのアクセサリー、柔らかいベルベットの背景。」
```

### ルール4: 目的・文脈を伝える

用途やターゲットを伝えると、モデルが論理的な判断をして適切な画像を生成する。

```
○ 「20代女性向けのInstagram広告用。春の新作コスメを訴求する、
    明るくフレッシュな雰囲気のビジュアル。」
```

## クリエイティブディレクターのように指示する

### ライティング（照明）制御

| 指示例 | 効果 |
|--------|------|
| 「3点ソフトボックスセットアップ」 | 均一にプロダクトを照らす |
| 「キアロスクーロ照明、高コントラスト」 | ドラマチックな明暗 |
| 「ゴールデンアワーのバックライト、長い影」 | 温かみのある逆光 |
| 「リムライト」 | 被写体の輪郭を際立たせる |
| 「自然光のみ、北向きの窓から」 | 柔らかく均一な光 |

### カメラ・レンズ指定

モデルは実在するカメラやレンズの名前を認識し、その特性を再現する。

| 指定 | 効果 |
|------|------|
| GoPro | 没入感のある歪んだアクション感 |
| Fujifilm | 本物のカラーサイエンス |
| 使い捨てカメラ | 生々しいノスタルジックなフラッシュ感 |
| 「ローアングル、浅い被写界深度（f/1.8）」 | ボケ味のある迫力ショット |
| 「広角レンズ」 | スケール感の表現 |
| 「マクロレンズ」 | 細部のディテール |
| 「望遠レンズ 200mm」 | 圧縮効果・背景ぼかし |

### 雰囲気・カラーグレーディング

| 指示例 | 効果 |
|--------|------|
| 「1980年代のカラーフィルムで撮影したような、やや粒子感のある」 | ノスタルジック・グリッティ |

| 「ハイキーライティング、白背景」 | クリーンなプロダクトショット |

## セクション別テクニック

### 1. テキスト描画・インフォグラフィック

nanobanana2はテキストの描画精度が非常に高い。

- テキストは **引用符（""）** で囲む
- フォントスタイルを明示する: 「太字のサンセリフ」「ネオンの筆記体」
- 「このデータを視覚的なインフォグラフィックに圧縮して」のような指示が有効
- スタイル指定: ポリッシュなエディトリアル / テクニカルダイアグラム / 手描きホワイトボード

### 2. キャラクター一貫性

- 最大14枚のリファレンス画像（うち6枚まで高精度）
- 「顔の特徴をリファレンス画像と同一に保ちながら、表情・ポーズ・環境を変える」と明示
- 複数キャラクターにはそれぞれ名前を付けて区別する

### 3. 高度な編集・修復

- マスキング不要。自然な文章で変更を指示
- 「観光客を消して、論理的なテクスチャで埋める」
- 季節・ライティングの変更も構造を維持したまま可能
- 漫画のカラー化、文化的ローカライズ（テキスト翻訳含む）にも対応

### 4. 2D ↔ 3D変換

- 間取り図 → フォトリアリスティックな3Dインテリアビジュアル
- フラットなイラスト → 立体レンダリング
- コンポジションを維持したまま次元変換

### 5. 高解像度・テクスチャ

- ネイティブ1K〜4K生成対応
- 表面の質感を詳細に指定: 「マットな仕上げ」「ブラッシュドスチール」「しわのある紙」
- 壁紙・大判印刷にも対応するピクセルパーフェクトな出力

### 6. ワンショット・ストーリーボード

- 9枚以上の連続画像でキャラクターの外見・服装の一貫性を維持
- アングルや感情のビートを変化させながらナラティブを構築
- 商業用途: 一貫した主人公による高級プロダクトのストーリー

### 7. 構図・レイアウト制御

- リファレンス画像をアップロードしてレイアウトを厳密にコントロール
- 手描きスケッチ → 要素配置の指定
- ワイヤーフレーム → 高精細UIモックアップ
- グリッドテンプレート → タイルベース・LED表示用画像
- スプライトシート（3x3グリッド）→ アニメーション素材

## プロンプトの5要素

効果的なプロンプトには以下の5つの要素を含める:

### 1. スタイル / メディウム（表現手法）

画像の表現形式を指定する。これが出力の美的印象を大きく左右する。

| 例 | 効果 |
|----|------|
| 写真 | フォトリアリスティックな出力 |
| イラスト | 手描き風の表現 |
| 水彩画 | 柔らかくにじんだ色彩 |
| 3Dレンダリング | 立体的・プロダクト向け |
| フラットデザイン | モダンでクリーンな印象 |
| 油絵 | 重厚でテクスチャのある表現 |

### 2. 主題（Subject）

誰が・何が画像に含まれるか。外見・服装・ポジションを詳細に描写する。

```
✕ 「男性」
○ 「30代前半の日本人男性。ネイビーのスーツにノータイ、
    自信に満ちた微笑みでカメラに目線を向けている」
```

### 3. 場所（Setting）

シーンの舞台を設定する。現実的な場所からファンタジーまで対応。

```
○ 「東京・渋谷のスクランブル交差点、雨上がりの夕暮れ」
○ 「白い大理石のテーブルの上、自然光が差し込むカフェ」
○ 「ミニマルな白い撮影スタジオ、影なし」
```

### 4. アクション（Action）

シーン内で何が起きているか。動き・インタラクションを描写する。

```
○ 「コーヒーカップを両手で包み込むように持ち、湯気を見つめている」
○ 「スマートフォンの画面を見せながら、驚いた表情で友人に話しかけている」
```

### 5. 構図（Composition）

カメラアングルやフレーミングを指定する。

| 構図指示 | 効果 |
|---------|------|
| クローズアップ | 表情・ディテール重視 |
| バストショット | 上半身、ポートレート向け |
| フルショット | 全身、環境との関係性 |
| 俯瞰（バードアイビュー） | 全体を見渡す |
| ローアングル | 迫力・権威感 |
| オーバーザショルダー | 臨場感・視聴者参加感 |

## 編集テクニック（生成後の調整）

一度生成した画像に対して、以下の編集指示が可能:

### キャラクター差し替え
主題を変更し、スタイルと環境はそのまま維持。

### 構図変更
同じシーンの別アングルを生成。

### アクション変更
キャラクターの行動を別のものに差し替え。

### 場所変更
環境のみを部分的または完全に変更。

### スタイル変更
写真 → イラスト → 漫画 → ユニークなアート表現に変換。

## 実践Tips

### Tip 1: テキストは引用符で囲む

```
○ 「"Happy Birthday" と書かれたケーキ。太字のサンセリフフォント。」
○ 「"春の新作" というネオンサインが光るショーウィンドウ」
```

### Tip 2: 翻訳・ローカライズ

任意の言語でコンセプトを書き、ターゲット言語と文化的参照を指定する。正確なテキストは引用符で指定。

### Tip 3: 実在するカメラ・フィルムの名前を使う

モデルは実在の機材名を認識し、その特性を再現する。

```
○ 「Hasselblad中判カメラで撮影したようなポートレート」
○ 「Kodak Portra 400フィルムで撮影した、柔らかい粒子感のある色味」
○ 「iPhone 16 Proのポートレートモードで撮影したような自然なボケ」
```

### Tip 4: 詳細を追加するほど精度が上がる

ミニマルなプロンプトでも高品質な結果は出るが、ディテールを追加するほど意図に近づく。

```
最小限: 「猫」
改善:   「窓辺に座る三毛猫」
最適:   「古い木製の窓枠に座る三毛猫。午後の日差しが差し込み、
         レースのカーテンが風にそよいでいる。Fujifilm X-T5で撮影、
         f/2.8、自然光のみ。」
```

### Tip 5: 反復的に改善する

完璧を一発で目指さない。まず生成し、その結果に対して「もう少し○○にして」と編集を重ねる。

### Tip 6: モデルの知識を活用する

歴史的な時代、実在の場所、概念的なフレームワークを参照すると、モデルの知識ベースが活用される。

```
○ 「1950年代のアメリカンダイナー風のインフォグラフィック」
○ 「バウハウススタイルのタイポグラフィポスター」
○ 「浮世絵風の現代東京の風景」
```

### Tip 7: 複数画像の一括生成

```
○ 「同じプロダクトの3つの異なるバリエーションを作成」
○ 「4つのカラーパレットで並べて比較」
```

## 広告クリエイティブ向けの実践プロンプト例

### Meta フィード広告（1200x628px）

```
横長のMeta広告バナー。アスペクト比1.91:1。
白い大理石のテーブルの上に、新作スキンケアセットが美しく配置されている。
3点ソフトボックス照明で均一に照らされたプロダクトショット。
右側に "今だけ30%OFF" のテキスト（太字のモダンサンセリフ、ダークネイビー）。
背景はソフトなピンクのグラデーション。清潔感と高級感のあるトーン。
```

### Meta ストーリーズ（1080x1920px）

```
縦型のストーリーズ広告。アスペクト比9:16。
20代の女性がカフェで新商品のドリンクを手に持ち、
カメラに向かって笑顔を見せている。自然光。
上部に "NEW" のテキスト（ネオンピンク、手書き風フォント）。
下部に "詳しくはこちら ↑" のCTA。
Instagram風のカジュアルで親しみやすい雰囲気。
```

### SNS投稿（1080x1080px）

```
正方形のSNS投稿画像。アスペクト比1:1。
ミニマルな白背景に、春の新作スニーカーが45度の角度で浮遊している。
ドロップシャドウで立体感を演出。
下部に "SPRING COLLECTION 2026" のテキスト（細身のサンセリフ、グレー）。
クリーンでモダンなプロダクトフォトグラフィー。Hasselblad風の描写。
```
